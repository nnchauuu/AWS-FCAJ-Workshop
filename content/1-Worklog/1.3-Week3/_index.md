### Week 3 Objectives:

* Learn about Amazon Virtual Private Cloud (VPC).
* Understand networking components such as Subnet, Route Table, Internet Gateway, and NAT Gateway.
* Distinguish between Security Groups and Network ACLs.
* Practice creating a basic network environment on AWS.
* Get familiar with VPC Flow Logs, EC2 Instance, Reachability Analyzer, and Session Manager.

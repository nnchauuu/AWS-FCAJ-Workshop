### Week 3 Objectives:

* Learn the basic concepts of Amazon Virtual Private Cloud (VPC).
* Understand VPC networking components such as Subnet, Route Table, Internet Gateway, and NAT Gateway.
* Understand how to control network access using Security Groups and Network ACLs.
* Practice preparing a network environment on AWS.
* Learn more about VPC Flow Logs, EC2 Instance, Reachability Analyzer, and AWS Site-to-Site VPN.

### Week 5 Objectives:

* Learn about AWS Cloud9 and browser-based development environments.
* Practice using the command line, text files, and AWS CLI in Cloud9.
* Learn about Amazon S3 and the object storage model.
* Practice hosting a static website using Amazon S3.
* Get familiar with Amazon CloudFront to accelerate website delivery.

### Week 5 Objectives:

* Learn about AWS Cloud9 and how to use a browser-based application development environment.
* Practice creating a Cloud9 instance and working with the command line and text files.
* Learn how to use AWS CLI in the Cloud9 environment.
* Learn about Amazon S3 and object storage.
* Practice hosting a static website using Amazon S3 and accelerating it with Amazon CloudFront.

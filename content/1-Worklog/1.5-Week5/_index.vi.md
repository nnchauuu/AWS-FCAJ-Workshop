### Kết quả đạt được tuần 5:

* Hiểu được vai trò của AWS Cloud9 trong việc phát triển ứng dụng trên môi trường đám mây.
* Biết cách tạo Cloud9 instance và thao tác với command line, file text trong Cloud9.
* Nắm được cách sử dụng AWS CLI cơ bản trong môi trường Cloud9.
* Hiểu được Amazon S3 là dịch vụ lưu trữ object và các thành phần như bucket, object, permission.
* Thực hành hosting website tĩnh bằng Amazon S3.
* Biết cách cấu hình public access, kiểm tra website và sử dụng CloudFront để tăng tốc website tĩnh.
* Hiểu thêm các thao tác quản lý object như versioning, di chuyển object, sao chép object sang region khác và dọn dẹp tài nguyên.
